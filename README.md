## 학습 내용

### 컴퓨터구조

#### 컴퓨터가 이해하는 정보, 컴퓨터의 네 가지 핵심 부품

컴퓨터가 이해하는 정보는 **데이터**와 **명령어**.

- **데이터**는 컴퓨터가 이해하는 숫자, 문자, 이미지, 동영상 등과 같은 정적인 정보
- **명령어**는 데이터를 움직이고 컴퓨터를 작동시키는 정보

컴퓨터의 네 가지 핵심 부품은 **CPU**, **메모리**, **보조기억장치**, **입출력장치**.

- **CPU (중앙처리장치)**
  - 컴퓨터의 두뇌에 해당하며, 메모리에 저장된 명령어를 읽어 들이고 해석하고 실행하는 부품
  - CPU 내부 구성 요소:
    - **ALU (산술논리연산장치)**: 덧셈, 뺄셈 같은 산술 연산과 AND, OR 같은 논리 연산을 수행하는 계산기 역할
    - **레지스터**: CPU 내부의 작은 임시 저장 장치. 프로그램 실행에 필요한 값들을 일시적으로 저장하며, 여러 개의 레지스터가 존재함
    - **제어장치 (CU)**: 제어 신호를 내보내 컴퓨터 부품들을 관리하고 명령어를 해석하는 장치. 메모리 읽기/쓰기 신호 등을 발생시킴

- **메모리 (주기억장치, RAM)**
  - 현재 실행되는 프로그램의 명령어와 데이터를 저장하는 부품
  - 프로그램이 실행되려면 반드시 메모리에 저장되어 있어야 함
  - 주소(Address) 개념: 메모리에 저장된 값의 위치는 주소로 알 수 있으며, 주소를 통해 원하는 위치에 접근 가능
  - **휘발성 저장 장치**: 전원이 꺼지면 저장된 내용이 사라짐
  - CPU는 메모리에 저장된 값을 빠르게 읽고 쓸 수 있음

- **보조기억장치**
  - 메모리는 전원이 꺼지면 데이터가 사라지므로, 전원이 꺼져도 보관할 프로그램을 저장하는 부품
  - **비휘발성 저장 장치**: 전원이 꺼져도 저장된 내용이 유지됨
  - 메모리보다 크기가 크고 가격이 저렴하지만, 접근 속도가 느림
  - 대표적인 보조기억장치: HDD (하드 디스크), SSD (솔리드 스테이트 드라이브), USB 메모리, DVD, CD-ROM 등
  - 보조기억장치에 저장된 프로그램을 실행하려면 메모리로 옮긴(적재한) 뒤 CPU가 처리함

- **입출력장치**
  - 컴퓨터 외부에 연결되어 컴퓨터 내부와 정보를 교환하는 장치
  - **입력장치**: 컴퓨터에 데이터를 전달하는 장치 (예: 키보드, 마우스, 마이크, 스캐너)
  - **출력장치**: 컴퓨터가 처리한 결과를 사용자에게 보여주는 장치 (예: 모니터, 스피커, 프린터)
  - 보조기억장치도 넓은 의미에서 입출력장치의 일종으로 볼 수 있음 (컴퓨터 외부에 연결되어 데이터를 주고받으므로)

#### 핵심 부품 간 연결: 시스템 버스 (메인보드)

위 네 가지 핵심 부품은 **메인보드**에 연결되며, 부품 간 정보를 주고받는 통로인 **버스(Bus)**를 통해 소통한다. 그 중에서도 컴퓨터의 네 가지 핵심 부품을 연결하는 가장 중요한 버스를 **시스템 버스**라고 한다.

- **주소 버스**: 주소를 주고받는 통로
- **데이터 버스**: 명령어와 데이터를 주고받는 통로
- **제어 버스**: 제어 신호를 주고받는 통로
